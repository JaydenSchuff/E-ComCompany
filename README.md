### Installation

If you wish to install and customize yourself follow this process

1. Clone the repo
   ```sh
   git clone https://github.com/JaydenSchuff/E-ComCompany
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. fill in .env file
